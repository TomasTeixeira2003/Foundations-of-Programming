# Buggy Cata Base Recovery Project

## Overview
This project consists of a set of functions designed to solve five independent tasks aimed at identifying and fixing issues in the **Buggy Cata Base (BDB)**, which has become corrupted due to unknown causes.

The BDB stores authentication information for system users and is currently incorrectly denying access to some registered users.

---

## Objectives
The project is divided into the following tasks:

1. **Documentation Correction**  
   Review and fix inconsistencies or errors in the existing documentation.

2. **Database PIN Discovery**  
   Determine the correct PIN required to access the database.

2. **Data Consistency Verification**  
   Check and ensure the integrity and coherence of the stored data.

4. **Content Decryption**  
   Decrypt the contents of the database to restore readable information.

5. **Password Debugging**  
   Identify and fix issues related to user passwords.

---

## Context
Due to corruption in the system, some valid users are being denied access. The goal of this project is to restore correct authentication behavior by addressing the issues listed above.
