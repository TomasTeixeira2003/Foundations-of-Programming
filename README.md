# Introduction to Computer Science Projects

This repository contains projects developed during an introductory Computer Science course focused on fundamental programming concepts such as algorithms, abstraction, and programming paradigms, using **Python**.

## Projects

- **Buggy Cata Base Recovery**  
  Fixing a corrupted authentication database (PIN discovery, data validation, decryption, and password debugging).

- **Meadow Ecosystem Simulation**  
  Simulation of a predator-prey ecosystem with movement, reproduction, and evolution over generations.
